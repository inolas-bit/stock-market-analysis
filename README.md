# 📊 Stock Market Data Analysis – Indian Stocks

## 🔍 Project Summary

Analyzed 2 years of Indian stock data (Reliance, TCS, Infosys) using Python and yfinance to compute:
- 📈 Moving Averages (20-day, 50-day)
- 📉 Daily Returns and Volatility
- 🔗 Correlation between companies

Visualized trends using Matplotlib and Plotly.

---

## 📁 Project Structure

```
stock-market-analysis/
│
├── stock_analysis.ipynb       # Jupyter notebook with all code
├── requirements.txt           # All dependencies
├── README.md                  # Project description
└── outputs/                   # Optional graphs or CSV exports
```

---

## 🛠 Tech Stack

- Python
- Pandas
- yfinance
- Matplotlib
- Plotly

---

## ✨ How to Run

1. Clone the repo
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Open `stock_analysis.ipynb` in Jupyter and run all cells
